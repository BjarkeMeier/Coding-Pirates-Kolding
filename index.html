Coding Pirates Kolding

Minecraft workshop af Bjarke Meier med materiale venligst udlånt af Christian Riis.

Det er en forudsætning, at du har en Minecraft klient og en Minecraft konto. Når du starter din Minecraft klient skal du vælge at køre version 1.9 (ikke 1.9.2 eller noget som helst andet). Det er den version serveren kører.

  1. Server - Download, unzip og kør startscript. På mac kør evt. kommandoen "chmod a+x FILENAME" for at gøre scriptet eksekverbart.
  2. Lektion 1 - Leg med det og slut af med at lave en kommando, der fx bygger en trappe.
  3. Lektion 2 - Leg med det og programmer din egen kommando, der fx bygger en trappe.

Lær mere:

  1. www.scriptcraftjs.org - Her er masser af god dokumentation og vejledninger (på engelsk)
  2. www.w3schools.com - Læs om javascript, men spring alt om HTML over (på engelsk)
  3. Events: Scriptcraftjs.org duer ikke til at se info om events. Se i stedet her.

Kommandoer

Eksempel på simpel kommando, der laver en blok.

    function block() {
      this.box0(blocks.stone, 1, 1, 1);
    }
    Drone.extend(block);

Eksempel på simpel kommando, der laver en blok. Kan sættes sammen med andre kommandoer som i fx /js block().up(1).right(1).block()

    function block(){
      this.box0(blocks.stone, 1, 1, 1);
      return this;
    }
    Drone.extend(block);

Eller det samme på en anden måde...

    function block(){
      var drone = this;
      drone.box0(blocks.stone, 1, 1, 1);
      return drone;
    }
    Drone.extend(block);

Parameter og for-loop (for-løkke)

    function trappe(hoejde) {
      // Hej med jer - dette er en kommentar
      for(var i = 0; i < hoejde; i = i + 1) {
        this.box(blocks.diamond, 1, 1, 1).right(1).up(1);
      }
    }
    Drone.extend(trappe);

Sjov med events

    events.blockBreak(function(event){
      echo( event.player, 'Hey - der var en gris i blokken!'); 
      spawn('PIG', event.block.location);
    });

3D print med Printcraft

  1. Download denne server. Unzip den og start den med et af de start_xx scripts du finder i den udpakkede mappe.
  2. Lav en kommando i Scriptcraft eller kopier en af dem du lavede i Scriptcraft øvelserne til denne server.
  3. Brug kommandoen til at bygge noget i en af de markerede firkanter
  4. Stil dig inde i firkantet og eksporter det du byggede med Minecraft kommandoen /print.
  5. Download den dannede print-fil og få Jens til at hjælpe dig med at print den :-)
